About ProjectManager

ProjectManager is a full-stack web application built using Angular (frontend) and Node.js / Express (backend), with secure JWT authentication and MongoDB as the database.

Features
  • User authentication with secure JWT tokens
  • Create, update, delete, and view projects
  • Clean UI with Bootstrap 5 and responsive layout
  • Protected routes and user-specific data
Tech Stack
  • Angular
  • Node.js
  • Express
  • MongoDB
  • Bootstrap 5
